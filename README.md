# 🌱 Harvest

Harvest is a modern marketplace application built with Flutter that connects rural producers directly with customers. The platform enables farmers and local producers to showcase and sell fresh vegetables, fruits, dairy products, and other locally produced goods, helping strengthen local economies while making fresh products more accessible.

## ✨ Features

* 🥕 Browse fresh vegetables and fruits
* 🌾 Explore locally produced products
* 🔍 Search and filter products
* 🛒 Marketplace experience for buyers and sellers
* 👤 User authentication
* 📝 Rich text product descriptions
* ❤️ Save favorite products
* 📱 Responsive and modern user interface
* 🌙 Light & Dark theme support

---

## 🛠️ Built With

* **Flutter** – Cross-platform UI framework
* **Riverpod** – State management
* **GoRouter** – Declarative navigation
* **Flutter DotEnv** – Environment configuration
* **Flutter Quill** – Rich text editor
* **REST API** – Backend communication

## 🚀 Getting Started

### Prerequisites

* Flutter SDK
* Dart SDK
* Android Studio or VS Code
* Android SDK / Xcode (for iOS)

### Installation

Clone the repository

```bash
git clone https://github.com/your-username/harvest.git
```

Navigate to the project

```bash
cd harvest
```

Install dependencies

```bash
flutter pub get
```

Create an environment file

```
.env
```

Example

```env
API_BASE_URL=https://your-api-url.com
```

Run the application

```bash
flutter run
```
